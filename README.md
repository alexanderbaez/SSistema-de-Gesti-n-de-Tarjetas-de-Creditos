# 🏦 Sistema de Gestión de Tarjetas de Crédito

Este proyecto es una **aplicación web simulada** desarrollada en HTML, CSS y JavaScript, que emula el funcionamiento de un sistema bancario para gestionar **tarjetas de crédito, clientes, resúmenes y pagos**. Está diseñado con interfaces diferenciadas para tres tipos de usuarios:

- 👤 Cliente
- 🧑‍💼 Agente de ventas
- 🧾 Operario del banco

## 🎯 Objetivo del Proyecto

Desarrollar una interfaz web estructurada y funcional para simular el flujo de trabajo real de un sistema de gestión de tarjetas de crédito, abarcando desde el alta de clientes hasta la gestión de resúmenes mensuales.

## 🧩 Tecnologías Utilizadas

- HTML5
- CSS3 (con diseño responsive)
- JavaScript (manejo de formularios y navegación)
- Git + GitHub para control de versiones

## 🗂️ Estructura del Proyecto

- `/HTML`: Contiene las diferentes vistas del sistema (login, cliente, agente y operario)
- `/CSS`: Hojas de estilo para la presentación y estructura responsive
- `/JAVASCRIPT`: Lógica para el control de navegación entre secciones
